# 360° Panorama Viewer

Three.jsで作った最小構成の360°パノラマビューアです。

## 使い方

### 推奨：ローカルサーバーで起動

このフォルダで次を実行してください。

```bash
python3 -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000
```

## 操作

- ドラッグ：視点移動
- マウスホイール：ズーム
- スマホの「モーション開始」：端末の向きで視点移動
- スマホの「VRゴーグル」：左右2分割表示に切り替えてゴーグルで表示
- 「パノラマ画像を開く」：別の画像に差し替え
- 「視点をリセット」：初期視点に戻す
- 「全画面」：全画面表示

## 画像について

360°パノラマ画像は通常、横:縦 = 2:1 の equirectangular 形式です。
このサンプルでは `assets/panorama.jpeg` を初期画像として読み込みます。

## ファイル構成

```text
index.html
style.css
main.js
assets/panorama.jpeg
```

## 注意

`index.html` を直接ダブルクリックして開くと、ブラウザの制限によりJavaScriptモジュールや画像読み込みが失敗する場合があります。
そのため、ローカルサーバーでの起動を推奨します。

スマホのモーション操作はブラウザのセンサー権限が必要です。端末によっては `https://` または `localhost` 以外では利用できない場合があります。
VR表示では横向き切り替え後に数サンプル待って正面を合わせます。ゴーグルに入れた状態で向きがずれる場合は、端末を正面に向けて「視点をリセット」を押してください。
